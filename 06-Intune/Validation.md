# Validation

## Objective

Verify the successful creation and assignment of Microsoft Intune policies.

---

## Validation Checklist

| Test | Expected Result | Status |
|------|-----------------|--------|
| Intune accessible | Admin Center available | ✅ |
| Device group created | Windows Managed Devices available | ✅ |
| Compliance policy created | Policy available | ✅ |
| Configuration profile created | Profile available | ✅ |
| Endpoint Security configured | Microsoft Defender settings configured | ✅ |
| Windows Update Ring created | Update Ring available | ✅ |
| Policies assigned | Assigned to Windows Managed Devices | ✅ |

---

## Validation Result

All Microsoft Intune policies were successfully created and assigned.

No Windows devices were enrolled during this phase; therefore, device-level policy application will be validated after endpoint enrollment in a future phase of the project.
