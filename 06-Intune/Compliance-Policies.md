# Compliance Policies

## Overview

A Windows compliance policy was created to ensure managed devices meet organizational security requirements before accessing corporate resources.

---

## Policy Information

| Property | Value |
|----------|-------|
| Policy Name | Corporate Windows Compliance Policy |
| Platform | Windows 10 and later |

---

## Compliance Settings

| Setting | Configuration |
|----------|---------------|
| Password Required | Yes |
| Minimum Password Length | 8 |
| BitLocker Required | Yes |
| Secure Boot Required | Yes |
| TPM Required | Yes |
| Microsoft Defender Required | Yes |

---

## Assignment

Windows Managed Devices

---

## Business Purpose

Compliance policies help organizations ensure that only secure and trusted devices can access Microsoft 365 services.
