# Microsoft Intune

## Overview

This section documents the deployment and configuration of Microsoft Intune within the Microsoft 365 Hybrid Security Lab.

Microsoft Intune was configured to establish enterprise endpoint management capabilities, including device compliance, configuration management, endpoint security, and Windows Update management.

At this stage of the lab, no Windows devices have been enrolled. The environment has been prepared for future device enrollment and policy deployment.

---

## Objectives

- Configure Microsoft Intune
- Create Azure AD device security groups
- Deploy Windows compliance policies
- Create Windows configuration profiles
- Configure Microsoft Defender security settings
- Deploy Windows Update Rings
- Assign policies to managed device groups
- Validate policy deployment
