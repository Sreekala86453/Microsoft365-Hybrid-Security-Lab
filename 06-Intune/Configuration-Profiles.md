# Configuration Profiles

## Overview

A Windows Settings Catalog profile was created to enforce security and user configuration settings across managed devices.

---

## Profile Information

| Property | Value |
|----------|-------|
| Profile Name | Corporate Windows Security Baseline |
| Platform | Windows 10 and later |
| Profile Type | Settings Catalog |

---

## Configured Settings

| Category | Setting | Value |
|----------|---------|-------|
| Microsoft Defender | Scan all downloaded files and attachments | Enabled |
| Microsoft Defender | Turn on behavior monitoring | Enabled |
| Microsoft Defender | Turn off real-time protection | Disabled |
| Control Panel | Prohibit access to Control Panel and PC settings | Enabled |

---

## Business Purpose

The configuration profile establishes a baseline security posture by enabling Microsoft Defender protections and restricting access to critical system settings.
