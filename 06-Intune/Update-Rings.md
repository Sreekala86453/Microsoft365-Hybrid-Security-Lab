# Windows Update Rings

## Overview

Windows Update for Business was configured to automate Windows updates while minimizing business disruption.

---

## Update Ring

| Property | Value |
|----------|-------|
| Name | Windows Update Ring - Standard |
| Product Updates | Enabled |
| Driver Updates | Allowed |
| Quality Update Deferral | 3 Days |
| Feature Update Deferral | 7 Days |
| Active Hours | 8:00 AM – 5:00 PM |
| Automatic Updates | Auto install and restart at maintenance time |

---

## Assignment

Windows Managed Devices

---

## Business Purpose

The Update Ring ensures devices remain secure while providing administrators with controlled deployment of Windows updates.
