# SharePoint Online

## Overview

This section documents the deployment and configuration of Microsoft SharePoint Online as part of the Microsoft 365 Hybrid Security Lab.

A Team Site was created to provide centralized collaboration, document management, and secure information sharing across multiple departments. The implementation demonstrates SharePoint administration, document library management, role-based access, and collaboration capabilities within Microsoft 365.

---

## Objectives

- Deploy a SharePoint Team Site
- Create and manage document libraries
- Organize departmental folders
- Upload and manage business documents
- Configure site permissions
- Publish company news
- Validate user access
