# SharePoint Online Screenshots

## Overview

This folder contains screenshots captured during the deployment, configuration, and validation of Microsoft SharePoint Online within the Microsoft 365 Hybrid Security Lab.

The screenshots provide visual evidence of the SharePoint Team Site creation, document management, collaboration features, permissions, and successful user validation.
