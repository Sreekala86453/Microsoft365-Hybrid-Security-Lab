# Document Libraries

## Overview

A dedicated document library named **Department Documents** was created to organize business documents for different departments.

The library provides centralized storage, collaboration, and document version management.

---

## Document Library

| Library | Description |
|----------|-------------|
| Department Documents | Central repository for departmental collaboration |

## Business Purpose

Each department maintains its own documentation while collaborating through a centralized SharePoint library.

Examples include:

- IT security policies
- Financial planning documents
- Employee onboarding documentation
- Corporate policies
- Project documentation
