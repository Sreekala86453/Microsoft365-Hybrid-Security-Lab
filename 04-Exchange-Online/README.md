# Exchange Online

## Overview

This section documents the deployment and administration of Microsoft Exchange Online within the Microsoft 365 Hybrid Security Lab.

The implementation includes:

- User mailbox provisioning
- Shared mailbox deployment
- Mailbox delegation
- Distribution Group management
- Exchange Online mail flow validation

The objective was to simulate a production-ready messaging environment supporting secure collaboration and centralized communication.

---

## Objectives

- Configure Exchange Online user mailboxes
- Deploy a shared Helpdesk mailbox
- Assign mailbox delegation permissions
- Create Distribution Groups
- Validate email delivery
- Document administrative procedures
