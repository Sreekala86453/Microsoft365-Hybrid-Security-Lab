# Benefits of Microsoft 365 E5

The Microsoft 365 E5 license provides enterprise-grade productivity, security, and compliance features including:

- Microsoft Entra ID
- Exchange Online
- SharePoint Online
- Microsoft Teams
- OneDrive for Business
- Microsoft Intune
- Microsoft Defender
- Microsoft Purview
- Microsoft Sentinel (integration)
- Conditional Access
- Multi-Factor Authentication (MFA)
