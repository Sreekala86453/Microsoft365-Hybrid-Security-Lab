# Microsoft 365 Tenant Configuration

## Overview

This section documents the initial configuration and validation of the Microsoft 365 E5 tenant used throughout this project.

The tenant serves as the cloud foundation for implementing Microsoft Entra ID, Exchange Online, SharePoint Online, Microsoft Intune, Microsoft Defender, Azure Monitor, and Microsoft Sentinel.

## Objectives

- Verify Microsoft 365 tenant information
- Validate Microsoft 365 E5 subscription
- Confirm tenant domain configuration
- Review available licenses
- Prepare the environment for Microsoft 365 administration

## Technologies

- Microsoft 365 Admin Center
- Microsoft Entra Admin Center
- Microsoft 365 E5 Trial
